# hampool

FCC Technician Class question pool (2026–2030) for interactive study sessions with [opencode](https://opencode.ai).

## Getting started

1. **Download opencode:**

   ```
   curl -sSf https://opencode.ai/install | sh
   ```

   Or follow the instructions at https://opencode.ai/docs/installation.

2. **Run it in this directory:**

   ```
   opencode
   ```

3. **Start studying:**

   Say **"quiz me"** to begin a quiz session. The built-in pool-quiz skill will walk you through random questions from the pool, track your progress, and explain answers.
